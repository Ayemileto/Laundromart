Customise Appearance

Some basic appearance customisations can be performed on the admin dashboard via Settings => Appearance Settings.
Some of such settings include changing the front and auth themes and customising the colours od the Dashboard theme.

Adding Custom Theme

Attached is a guide to add custom themes of your own.

  • Head over to app\views.
  • Select the appropriate folder for the theme you want to add.
    • auth for Authentication theme
    • backend for admin/staff/user dashboard theme
    • email for email themes (used when sending emails)
    • frontend for landing page and other public pages.
  • Create a folder for your new theme in the selected folder above. the folder name must end with "_theme"
  • Copy, Paste and Modify the layout.php file of one of the other themes to match what is applicable for your theme.
  • Using one of the existing themes as a guide, create all the neccessary subfolders and files, and apply your new themes styling.
    Hint: It might be easier to copy the files from the other themes, and edit instead
  • To use your new theme, head over to Admin dashboard => Settings => Appearance Settings and select your theme from the appropriate dropdown option.
