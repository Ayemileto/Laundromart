Basic Setup

The Staff system enables you to ass users as staffs and assign roles and permissions to them depending on how your organisation is structured.

The system requires you to first create roles with different permissions, then you can assign this roles to different users. As an example, you may have roles like Accountants - which will have access to all transaction records, Dispatch Rider - which will have access to the Pickup/Delivery scedules among other roles.

Creating Roles and Permission

Head over to Admin dashboard

  • Click "Staffs" menu, then "Roles and Permissions".
  • Click Add Role to add a role.
  • Insert the name of the role in the "Name" field.
  • Select the Appropriate permissions for the role.
  • Save the role and permissions.

You can use this method to create as many roles and permissions as you need.

Create Staffs

Once you create the roles and permissions you need, The next thing is to add staffs and assing roles to them.

  • On Admin dashboard, Click "Staffs" menu, then "Staffs List".
  • Click Add Staff.
  • Type the email of the staff to search (The staff is expected to have created an account).
  • Select the appropriate user from the dropdown.
  • The roles you previously created will show in the roles dropdown. select the appropriate role.
  • Save the Staff.
  • The newly created staff will have all the permissions assigned to the role.

Congratulations, Your have successfully creted a staff and assigned role and permissions!!!
